## نسخه‌ی غیر رسمی کلاینت رستم مدیا

رستم مدیا راهی برای دانلود محتوا از راه رسیور های ماهواره‌ایه. رستم مدیا یک اینترنت یک طرفه را از طریق جاساز کردن اطلاعات داخل بسته های تی‌اس منتقل میکنه. برای استخراج اطلاعات شما باید از شبکه‌ی ماهواره‌ای رستم مدیا به مدت چهار ساعت ضبط کنید. سپس با برنامه استخراج از فایل تی‌اس محتوا رو استخراج کنید. رستم مدیا فعلا تنها روی ویندوز کار میکنه. این کلاینت رو برای بقیه ی سکوها با زبان سی‌پلاس‌پلاس نوشته شده.
